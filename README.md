# AXI4LITE Doğrulama IP'si (VIP)

## Genel Bakış
Bu depo, AXI4LITE tabanlı tasarımları doğrulamak için geliştirilmiş AXI4LITE Doğrulama IP'sini (VIP) içermektedir. VIP, kapsamlı bir doğrulama sağlamak için gerekli tüm bileşenleri (ajanlar, sürücüler, monitörler ve skor tabloları) içerir.
## RTL Kaynağı

`axi4_lite_slave.sv` dosyası şu GitHub projesinden alınmıştır: [Axi4-lite](https://github.com/arhamhashmi01/Axi4-lite).

Bu dosya doğrulama sürecinde kullanılmış ve test edilmiştir. Lisans bilgileri için kaynak projeye başvurabilirsiniz.

## Coverage
AXI4LITE doğrulama sürecinde kapsam (coverage) düzenlenecek ve eksik senaryolar belirlenecektir. Bu, doğrulama sürecinin etkinliğini artırmak için önemli bir adımdır.
